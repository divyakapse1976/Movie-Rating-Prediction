# Movie Rating Prediction

This project predicts movie ratings based on various features such as
year of release, duration, genre, votes, director, and lead actors.
The goal is to build a machine learning model that can estimate a
movieâ€™s IMDb rating.

Project Objective
To develop a regression model that predicts movie ratings using
historical movie data and machine learning techniques.

Dataset
- Source: IMDb Movies India Dataset
- Features Used:
  - Year
  - Duration
  - Votes
  - Genre
  - Director
  - Actor 1
  - Actor 2
  - Actor 3
- Target Variable:
  - Rating

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

Project Workflow
1. Data Loading and Exploration  
2. Data Cleaning and Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Encoding  
5. Train-Test Split  
6. Model Training (Random Forest Regressor)  
7. Model Evaluation  
8. Sample Prediction  

Model Evaluation
The model performance is evaluated using:
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

How to Run the Project
1. Clone the repository  
2. Install required libraries  
3. Open the Jupyter Notebook  
4. Run cells sequentially  

Result
The trained model successfully predicts movie ratings with reasonable
accuracy based on available features.
